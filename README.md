# Random Pair Table
#### The project was created with Python 3
# Run
#### Create Virtualenv and install all required packages:
```bash
  $ virtualenv -p python3 env
  $ source env/bin/activate
  $ pip install -r requirements.txt
```
#### Run project:
```bash
  $ python main.py
```
